## Installation

    cd backend
    bundle install
    bundle exec rake db:setup
    bundle exec rails s

    cd frontend
    npm install
    bower install
    ember server --proxy=http://localhost:3000

Open a browser and navigate to
[http://localhost:4200](http://localhost:4200).
